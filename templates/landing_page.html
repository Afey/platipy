Chrome extension for readthedocs documentation

About

A chrome extension that allows for favoriting, commenting, and adding example code inline to documentation hosted by Read the Docs.

This extension is currently available for use on the following sites:

  1. docs.python.org
  2. flask.readthedocs.org>
  3. django.readthedocs.org>
  4. jquery-xml-rpc.readthedocs.org/
  5. oauthlib.readthedocs.org

Written in Python, Javascript, jQuery, HTML and CSS

Enables users to log into github via Oauth2

Uses the Pygments python library for syntax highlighting

Install

  1. Download this zip file
  2. Go to chrome://extensions
  3. Check "developer mode" in the upper right hand corner
  4. Click "Load unpacked extension" and browse to select the file
